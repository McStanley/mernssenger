## Tech stack

**Front End:** React, React Router, React Hook Form, SWR, Tailwind

**Back End:** NodeJS, Express, MongoDB, PassportJS

**Language:** TypeScript

## Usage

### Install

```sh
npm install
```

### Environment variables (`.env`)

```toml
MONGODB_URI="<mongodb_connection_string>"
SESSION_SECRET="<express-session_secret>"
```

### Run scripts

```sh
npm run build
npm run start
# OR
npm run dev
```
